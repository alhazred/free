free
====

Display amount of free and used memory on the illumos

#### Usage
<pre>
Usage: free [-p]
Display the amount of free and used system memory
</pre>
Example
<pre>
# free
           total       used       free     locked     kernel     cached
Mem:        128G       365M       107G          0      20.3G      6.87G
Swap:      95.6G      1.08G      94.5G
</pre>
